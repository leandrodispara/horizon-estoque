Como funciona o acesso

1. Gere ou defina um código para o cliente.

2. Envie o código e o link do sistema para ele.

3. O cliente usa em até 2 dispositivos simultaneamente.

4. Para bloquear por inadimplência, basta desativar o toggle na lista de clientes.

5. O bloqueio é imediato — a próxima requisição do cliente retorna acesso negado.
